# Contexto visual para IA y equipo

Añade aquí capturas de pantalla del sitio actual para que Cursor/IA tenga referencia visual sin alterar estructura ni textos.

## Archivos recomendados

| Archivo | Descripción |
|---------|-------------|
| `home.png` | Página de inicio (hero, secciones principales) |
| `map.png` | Mapa donde conviven historias y noticias en vivo |
| `upload-modal.png` | Modal o pantalla de subida de historias (texto/audio/video) |

Saca las capturas del sitio almamundi.org en producción o del clone local. Así la IA puede aplicar mejoras visuales manteniendo la estructura y el contenido.
